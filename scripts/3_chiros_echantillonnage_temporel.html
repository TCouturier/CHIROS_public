Modle jags pour estimer tendances

Sur la base des tirages de site raliss aprs application des filtres, nous avons tir 100 sites avec pondration par leur taille, puis lanc un modle destimation des taux de croissance.

Reconstitution du jeu de donnes

A partir des effectifs prdits sur chaque site/anne obtenus partir de ce modle (mdiane des postrieurs), nous avons rcupr la diffrence entre ces effectifs prdits et ceux observs (comptages rels) afin de calculer une variance rsiduelle moyenne sur chaque site. A partir de cette variance estime et de la prdiction moyenne de chaque site/anne, nous avons tir des valeurs dans une loi lognormale. Le choix de cette loi de distribution permettait de ne pas tirer des valeurs en des-sous de zro, comme cela aurait t le cas avec une loi normale. Ces valeurs estimes nous ont servi reconstituer les donnes manquantes dans les sries temporelles. Nous avons conserv les comptages rels lorsquils existaient.

Taux de croissance estims selon les modalits dchantillonnage temporel

On enregistre de fortes diffrences entre les espces en termes de prcision et de biais sur le taux de croissance lambda. Nous avons class les espces selon la prcision des lambda obtenue : Au dbut les espces avec mauvaise prcision, la fin les espces dont la prcision est trs bonne.

Graphes A,B : passer une anne sur deux diminue la prcision, et gros biais selon quelle anne on slectionne pour le A.
Graphe C : ok si une anne sur deux, forte imprcision une anne sur trois pour un des tirages (mais pas trop biais).
Graphe D : prcis une anne sur deux (mais un peu biais), forte imprcision une anne sur trois.
Graphe E : Forts biais selon les tirages (mme une anne sur deux), prcision reste bonne.
Graphes F, G : On reste prcis et pas trop biaiss une anne sur deux (F un peu biais), biais pour un tirage dune anne sur trois. Graphes H, I : On reste prcis et pas trop biaiss, mme avec une anne sur trois.

Plusieurs hypothses pour expliquer une faible prcision pour certaines espces : De fortes variances moyennes de comptages entre sites, des abondances trop faibles, et/ou des variations importantes dabondance sur les gros sites.
